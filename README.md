# Menagerie

A static site generator that supports:

- Markdown
- HTML
- JSON Schemas (Limited)
- XML Schemas
- YAML Schemas

# Documentation

Documentation for how to use menagerie is located at [https://bwc9876.github.io/menagerie/](https://bwc9876.github.io/menagerie/)
